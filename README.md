# PHP AJAX CRUD with PayPal Integration

## Features
- Create Product
- Read Products
- Update Product
- Delete Product
- AJAX without page refresh
- Bootstrap UI
- PayPal Sandbox Integration

## Technologies
- PHP
- MySQL
- AJAX
- Bootstrap 5
- PayPal REST SDK

## Installation

1. Clone the repository.
2. Import `schema.sql` into MySQL.
3. Update `db.php`.
4. Configure PayPal Sandbox credentials.
5. Run using XAMPP.
